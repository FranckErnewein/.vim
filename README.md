# .vim

vim configuration and plugin dependencies management based on pathogen and git submodules

## Installation:

```
git clone git://github.com/FranckErnewein/.vim ~/.vim
```

Create symlinks:

```
ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/gvimrc ~/.gvimrc
```

Start vim then 
```
:PlugInstall
```
